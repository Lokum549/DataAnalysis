# [cite_start]Лабораторна робота №2: Наука про дані: підготовчий етап [cite: 1, 2]

## Опис завдання

[cite_start]Ця лабораторна робота спрямована на опанування інструментів Python (Pandas, Jupyter, venv) для автоматизованого збору, очищення, статистичного аналізу та передобробки структурованих наборів даних[cite: 29, 35].

* [cite_start]**Частина 1:** Охоплює автоматизацію процесу збору, очищення та структурованої обробки даних про індекс VHI з вебресурсів за допомогою модулів `pandas` та `urllib`[cite: 28, 30, 35].
* [cite_start]**Частина 2:** Зосереджена на глибокому аналізі та підготовці масиву даних про споживання електроенергії (Individual Household Electric Power Consumption Dataset)[cite: 46, 48]. [cite_start]Включає складну фільтрацію за критеріями, профілювання часу виконання коду через модуль `timeit`, стандартизацію/нормалізацію та One Hot Encoding[cite: 47, 50, 55, 57]. 

[cite_start]Обидві частини виконані у вигляді окремих `.ipynb` файлів[cite: 58].

## Вимоги до системи

* **Операційна система:** Windows / macOS / Linux
* **Версія інтерпретатора:** Python 3.9 або новіше
* [cite_start]**Менеджер пакетів:** `pip` [cite: 9]
* [cite_start]**Залежності:** Всі необхідні бібліотеки (`pandas`, `jupyter` тощо) вказані у файлі `requirements.txt`[cite: 29].

## Інструкція із запуску

1. Відкрийте термінал у теці `lab_2`.
2. [cite_start]Створіть та активуйте віртуальне середовище[cite: 29]:
   * На Windows: 
     `python -m venv venv` 
     `venv\Scripts\activate`
   * На macOS/Linux: 
     `python3 -m venv venv` 
     `source venv/bin/activate`
3. Встановіть усі необхідні залежності командою:
   `pip install -r requirements.txt`
4. [cite_start]Для успішного виконання "Частини 2" потрібно завантажити файл датасету *Individual Household Electric Power Consumption* та покласти розпакований файл у теку `lab_2`[cite: 48].
5. Запустіть Jupyter Notebook у терміналі:
   `jupyter notebook`
6. [cite_start]Відкрийте файли `.ipynb` (наприклад, `Part1NOAA.ipynb` та `Part2.ipynb`) і послідовно виконайте комірки[cite: 43, 58].
