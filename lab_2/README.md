# Лабораторна робота №2: Наука про дані: підготовчий етап

### Опис завдання
Ця лабораторна робота присвячена роботі з великими масивами даних, їх очищенню, трансформації та статистичному аналізу. Робота складається з двох основних частин:

### Частина 1: Дані VHI (Vegetation Health Index)
* Автоматичне завантаження та очищення CSV-файлів з даними про стан рослинності в областях України.
* Перетворення індексів областей NOAA на український алфавітний порядок.
* Реалізація функцій для формування вибірок (пошук екстремумів, фільтрація за роками та областями).

### Частина 2: Енергоспоживання домогосподарств (Individual Household Electric Power Consumption)
* Робота з великим датасетом (>2 млн записів).
* Профілювання часу виконання операцій за допомогою модуля `timeit`.
* Фільтрація даних за складними умовами (потужність, сила струму, групи споживання).
* Нормалізація (`MinMaxScaler`) та стандартизація (`StandardScaler`) даних.
* Кореляційний аналіз (Пірсон та Спірмен).
* Створення нових категоріальних ознак ("День тижня", "Робочий/Вихідний") та застосування **One Hot Encoding**.

---

## Вимоги до системи
* **Операційна система:** Windows / macOS / Linux
* **Версія інтерпретатора:** Python 3.9 або новіше
* **Менеджер пакетів:** `pip`
* **Залежності:** Обов'язкові бібліотеки (`pandas`, `scikit-learn`, `jupyter`), усе вказано у файлі `requirements.txt`.
---

## Інструкція із запуску

### 1. Вимоги до середовища
1. Відкрийте термінал у теці `lab_2`.
2. Створіть та активуйте віртуальне середовище у цьому терміналі:
   python -m venv venv
   venv\Scripts\activate
3. Встановіть необхідні залежності:
   pip install -r requirements.txt
4. Для Частини 2: завантажте файл `household_power_consumption.txt` з репозиторію UCI та покладіть його в корінь теки `lab_2'.
5. Запустіть Jupyter Notebook:
   jupyter notebook
6. Відкрийти файли `Part1.ipynb` та `Part2.ipynb` і виконати комірки.
