# Лабораторна робота №3: Візуалізація багатовимірних даних 

## Опис завдання

Ця лабораторна робота присвячена дослідницькому аналізу даних, глибокому очищенню датасетів та побудові складних багатовимірних візуалізацій за допомогою бібліотек Matplotlib та Seaborn. 

Для виконання роботи обрано унікальний датасет **Heart Disease (підмножина Cleveland)** з репозиторію машинного навчання UCI. Робота включає наступні етапи:

**Очищення даних (Data Cleaning)**
* Завантаження "сирих" даних безпосередньо з сервера з автоматичним іменуванням колонок.
* Виявлення нестандартних пропущених значень (закодованих символом `?`) у числових атрибутах.
* Заміна пропущених значень на медіану для збереження цілісності даних та перевірка на наявність дублікатів.

**Побудова візуалізацій (9 графіків)**
* **Базові графіки:** Діаграма розсіювання (Scatter plot) для аналізу пульсу, лінійний графік (Line plot) для тренду холестерину та гістограма розподілу пацієнтів за 5 суворими віковими діапазонами.
* **Багатовимірні графіки (за статтею з Medium):** Побудова матриці взаємозв'язків (Pairplot), 4D-аналіз (Scatter plot з використанням кольору та розміру точок) та 5D-аналіз (FacetGrid з розбиттям на підграфіки за категоріями).
* **Статистичний аналіз:** Побудова теплової карти кореляції (Heatmap), ящиків з вусами (Boxplot) для пошуку аномалій у рівні холестерину та стовпчастої діаграми (Countplot).

## Вимоги до системи

* **Операційна система:** Windows / macOS / Linux
* **Версія інтерпретатора:** Python 3.9 або новіше
* **Менеджер пакетів:** `pip`
* **Залежності:** Обов'язкові бібліотеки (`pandas`, `matplotlib`, `seaborn`, `jupyter`), усе вказано у файлі `requirements.txt`.

---

## Інструкція із запуску

1. Відкрийте термінал у теці `lab3`.
2. Створіть та активуйте віртуальне середовище у цьому терміналі (приклад для Windows): `python -m venv venv` та `venv\Scripts\activate`
3. Встановіть необхідні залежності: `pip install -r requirements.txt` *(або `pip install pandas matplotlib seaborn jupyter`, якщо файлу немає)*.
4. *Примітка:* Файли з даними завантажувати не потрібно, скрипт автоматично витягує актуальний датасет з сервера UCI під час виконання першої комірки.
5. Запустіть Jupyter Notebook: `jupyter notebook`
6. Відкрийте файл `.ipynb` із лабораторною роботою та виконайте комірки по черзі.
